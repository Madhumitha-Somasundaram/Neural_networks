# Neural Networks & Language Models

This repository contains implementations of neural network models, including gradient descent, character-level language modeling, and bigram language modeling.

---

## Repository Structure

| File / Notebook                        | Description                                                                 |
|----------------------------------------|-----------------------------------------------------------------------------|
| `Gradient_Descent.ipynb`               | Jupyter notebook demonstrating gradient descent on example functions.      |
| `MLP_charater_level_language_model.ipynb` | Character-level neural network language model using MLP.                   |
| `bigram_language_model.py`             | Python implementation of a bigram-based language model.                     |
| `input.txt`                            | Input text file for training the language models.                           |
| `names.txt`                            | Dataset of names used in training or examples.                              |
| `train_gpt.py`                         | Python script for training GPT-style models.                                |
